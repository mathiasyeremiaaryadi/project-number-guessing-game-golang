## Run Locally

Clone the project

```bash
https://github.com/mathiasyeremiaaryadi/project-number-guessing-game-golang.git
```

Go to the project directory

```bash
cd my-project
```

Install dependencies

```bash
go build
```

Start the server

```bash
./guessing-game
```
